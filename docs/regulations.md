# Regulations Storage on Disk

## General Description
  The format of the Regulations API is split up into ... `agencies`, `dockets`, `docket`, `attachment`

* **Agency**
    * **Dockets**
        * Docket Description 
            * Stored as JSon
        * **Documents**
            * Attachments
                * PDF
                * HTM
            * **Comments**
                * Attachments
